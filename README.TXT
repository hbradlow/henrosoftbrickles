Dependencies:
	Lightweight Java Game Library - http://lwjgl.org/
	Slick2D - http://slick.cokeandcode.com/
	Phys2D - http://phys2d.cokeandcode.com/

Compiling:
	Place the lwjgl native libraries and other dependencies
	in $JAVA_HOME/jre/lib/ext/ and build with "ant jar"
